Get the game from Captain Zedo's website

Installation

  • Insert CD1
  • Run diskspace_skip.exe
    • Insert the drive letter and press Enter
  • Install the game in C:\Program Files\Hardwar
IMPORTANT: do not change the install directory

Patching

  • Run HardwarUIM06.exe and install the patch
  • Move HARDLUCK.EXE to the game directory and run it
    • Check the Test/cheat functions enabled box
  • Run registry_patch.reg
  • Insert both CDs
    • Open CD1 and copy all the .gdv files to the game directory
    • Repeat for CD2 and choose Don't copy when prompted to overwrite the files
Enjoy the game.

To install the game in a different directory:

  • Open registry_patch.reg with a text editor
    • Replace all "C:\\Program Files\\Hardwar\\" with your desired directory (don't forget the double slashes!)
